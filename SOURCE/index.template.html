Welcome, wizard amigos.

Come, learn to code with us.

Wizard Amigos Institute is a co-learning programming class. It is an open, friendly and collaborative space for girls and boys to learn how to code. Folks from communities that have historically been excluded from tech are very much encouraged to apply!

Magic Spells

We cover the main elements that every wizard apprentice needs to know:

  • Operating System
    • unix & bash
  • Markup Languages
    • markdown, html & css
  • Tools
    • chrome dev tools & atom.io
  • Collaboration
    • github, gitter, waffle & co.
  • Programming
    • javascript, nodejs & regex
  • DevOps
    • npm, git & browserify

Magic is serious business. There will be homeworks, so the learners will be able to practice their spells. To help them, we will produce a simple video summary after each session.

We also encourage all the participants to have some project they want to work on (but if they don't, no worries, will help them find one).

Classes of Magic

Wizard Amigos Institute will open its doors in May 2015. Every class will accept up to 10 learners and will be run by at least 2 higher-level wizards.

Unlocking Charm

There are some requirements for Classes of Magic:

You need to have your own computer. It doesn't have to be super powerful, but you need one.

It needs to run on Mac OSX or Linux. If you have Windows, we will help you install Linux.

You should have basic computer skills and feel comfortable finding information on the internet.

Most of the modern programming materials, tutorials and other online resources are written in english, so you should understand the basics. Mentors speak german and english, so they can help you out.

Who is Wizard Amigos Institute for

It is for girls and boys who want to learn special magic programming wizardry.

This is not a typical school. It is a Wizard Institute that promotes collaboration and a learning-by-doing approach. We will help you get to know the right magic spells and prepare you to become an active problem solver.
